# Loan-Details-Analysis-and-Performance-Dashboard

## Project Overview

Interactive Power BI dashboard analyzing a ₹1.30Cr loan portfolio (2023-2026). Features advanced DAX measures, Sync Slicers, and time-series trend analysis to evaluate default rates and optimize financial decision-making.


## The main objectives of this project are:

•	To analyse loan distribution across different states and purposes.

•	To evaluate loan approval and rejection patterns.

•	To study the relationship between loan amount and interest rate.

•	To identify trends in loan origination over time.

•	To create an interactive dashboard using Power BI for better decision-making

## Key Project Insights

## Financial Risk & Portfolio Health

## High-Risk Exposure: 
Identified a 25.2% Default Rate, primarily concentrated in loans with terms exceeding 36 months, suggesting a need to tighten long-term credit policies.

## Interest Rate Correlation:
Analysis revealed that loans with an interest rate above 15% saw a 12% higher delinquency rate, indicating a "tipping point" for borrower repayment capacity.

## Recovery Efficiency:
The Closure Rate stands at 23%, signaling an opportunity to optimize collection strategies for "Current" loans before they transition to "Default" status.
